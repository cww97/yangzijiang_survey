# 问卷机器人

## 使用说明

1. 打开 `config.ini`，修改 `url` 为问卷的链接（`repeat`为次数自定义）
2. 按住shift在空白处鼠标右击，点“在此处打开PowerShell窗口”
3. 敲命令 ` > python main.py `
4. 喝杯茶

### 如需要指定答案填写 
- `config.ini` 内的 `useans` 改为 `1`(否则填0)
- `answer` 填答案（逗号分隔，注意格式）
- 终端内有答案的日志，如有错误注意检查答案是否填写正确
